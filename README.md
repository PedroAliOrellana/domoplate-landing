# Domoplate Landing

A modern, mobile-first landing page built with Astro, React, Tailwind CSS v4, and Material Design 3.

## Features

- **Mobile-First Design**: Responsive layout scaling from 360px to 1440px+.
- **Material Design 3**: Custom styles implementing MD3 tokens.
- **Components**: Reusable MD3 Buttons, Cards, and Responsive Navbar.
- **Infrastructure**: Configured for Firebase Hosting with GitHub Actions.

## Getting Started

1. `npm install`
2. `npm run dev`

## Deployment

Push to `main` to deploy to Firebase Hosting (requires configuring secrets).
